# Catálogo de métricas

Páginas de referência para as métricas do painel **Central da Equipe Dados**, agrupadas pelas seções do dashboard. Cada página descreve o que cada métrica mede, qual tabela alimenta o cálculo e qual a frequência de atualização.

## Seções

- [Panorama de Dados](panorama-de-dados.md) — volumes, contagens de datasets/tabelas, status de publicação
- [Atualização](atualizacao.md) — proporção de tabelas, pipelines e tabelas semi-automatizadas por status
- [Prefect](prefect.md) — flows com schedule ativa
- [Custos](custos.md) — séries históricas de gasto com a GCP em USD e BRL

> Para adicionar uma métrica nova, use o [template de referência](https://github.com/basedosdados/manual-equipe-dados/blob/main/docs/_templates/referencia.md).
